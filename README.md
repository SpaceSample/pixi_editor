# PIXI Editor

## Env Setup
 - Node js 12+: See https://nodejs.org/en/
 - Yarn: ```npm install yarn```

## How to run?
```sh
yarn start
```

## Useage

### Tree View
 - press "Del" or "Backspace" to delete node
 - press Ctrl + "⬆️"/"⬇️" to change node order
 - (TODO) press Ctrl + "⬅️"/"➡️" to change node level
